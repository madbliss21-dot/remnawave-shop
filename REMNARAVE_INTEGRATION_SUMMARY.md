## ⚠️ Важные моменты

1. **Переменные окружения**
   ```bash
   REMNAWAVE_USERNAME=your_username
   REMNAWAVE_PASSWORD=your_password
   REMNAWAVE_API_URL=https://your-panel.com
   REMNAWAVE_SUBSCRIPTION_PATH=/sub
   ```

2. **Безопасность**
   - Никогда не коммитьте учетные данные в Git
   - Используйте HTTPS для API
   - Регулярно меняйте пароли

3. **Совместимость**
   - Требуется RemnaWave v2.15+
   - Python 3.8+
   - httpx >= 0.24.0
